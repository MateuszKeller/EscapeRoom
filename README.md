# EscapeRoom

Project for Engineering Thesis - Escape room game prologue design as a coherent set of selected mechanics defining the gameplay.

### Description
The game is in a style inspired by the horror stories of H.P. Lovecraft. At the beginning, the player in the lobby like room is introduced to the basic mechanics of the game. Then, in each of the 3 remaining rooms, he has to solve puzzles, which unlocks the passage to the next rooms. The final room has various solutions that determine the ending of the game.

Gameplay - https://youtu.be/Vpdnz-l10Ow

## Features

- Interaction system
- Object inspection
- Inventory
- Eyepiece (item showing hidden elements)
- Items to pick up.
- Dozen of different puzzles and interactable items
- Player UI and pop-ups
